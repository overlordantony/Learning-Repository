# 🧠 Learning Repository

Este repositorio es mi espacio personal de aprendizaje, práctica y experimentación como desarrollador.

Aquí documento mi proceso de reaprendizaje, reforzando fundamentos, explorando conceptos más avanzados y construyendo pequeños proyectos en distintos lenguajes y tecnologías.

---

## 🎯 Objetivo

El objetivo principal no es solo practicar, sino **entender profundamente** cómo funcionan las cosas.

Este repositorio está diseñado para:
- Reforzar bases de programación
- Explorar nuevas ideas y enfoques
- Experimentar sin miedo a romper cosas
- Documentar aprendizajes reales (incluyendo errores)

---

## 📂 Estructura

El repositorio está organizado en diferentes secciones:

### `fundamentals/`
Ejercicios enfocados en lógica, algoritmos y estructuras de datos.

### `labs/`
Espacio para pruebas, experimentos y exploración libre.

### `projects/`
Mini proyectos donde aplico lo aprendido en contextos más reales.

### `notes/`
Apuntes personales, conceptos clave y reflexiones técnicas.

### `mistakes/`
Errores, bugs y problemas interesantes junto con sus soluciones.

---

## 📌 Filosofía

Este repositorio no busca la perfección, sino el progreso.

Aquí hay:
- Código en proceso
- Ideas incompletas
- Experimentos fallidos
- Soluciones mejorables

Pero todo tiene un propósito: **aprender haciendo**.

---

## 🚀 Notas finales

Este es un proyecto vivo que irá creciendo con el tiempo.

Si en algún momento algo parece confuso, probablemente es porque estaba aprendiendo justo en ese momento… y eso también hace parte del proceso.
