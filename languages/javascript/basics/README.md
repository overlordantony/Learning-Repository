# JavaScript / Basics

Repaso completo de los fundamentos de JavaScript.  
Cada ejercicio tiene su propio código, notas teóricas y registro de bugs.

## Ejercicios

| # | Tema | Conceptos clave |
|---|------|-----------------|
| 01 | Variables | `var`, `let`, `const`, hoisting |
| 02 | Tipos de datos | primitivos, typeof, coerción |
| 03 | Operadores | aritméticos, comparación, lógicos, ternario |
| 04 | Condicionales | if/else, switch, nullish coalescing |
| 05 | Bucles | for, while, do-while, for...of, for...in |
| 06 | Funciones | declaración, expresión, arrow, parámetros, return |
| 07 | Arrays | métodos esenciales, iteración, spread |
| 08 | Objetos | propiedades, métodos, destructuring |
| 09 | Scope y Closures | scope léxico, closures, IIFE |
| 10 | DOM Básico | selección, eventos, manipulación |

## Cómo usar esto
1. Lee el `notes.md` del ejercicio antes de escribir código
2. Intenta resolver el ejercicio por tu cuenta
3. Si aparece un bug interesante, anótalo en `bugs.md`
4. El archivo de solución está como referencia, no como trampa

## Convención de archivos
```
01-variables/
├── exercise.js     ← tu código
├── notes.md        ← teoría y contexto
└── bugs.md         ← errores comunes y cómo resolverlos
```
