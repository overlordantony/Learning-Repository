# JavaScript / Intermediate

Conceptos intermedios de JavaScript: el salto entre "escribo código que funciona"
y "entiendo por qué funciona y cómo estructurarlo bien".

## Ejercicios

| # | Tema | Conceptos clave |
|---|------|-----------------|
| 01 | Array methods avanzados | flatMap, reduce compuesto, sort estable, Array.from |
| 02 | Objetos avanzados | getters/setters, computed props, Object.create, inmutabilidad |
| 03 | Prototypes y clases | prototype chain, ES6 classes, herencia, static, private fields |
| 04 | Async: callbacks y promises | event loop, callbacks, Promise, then/catch/finally, Promise.all |
| 05 | Async/Await | async/await, manejo de errores, paralelismo vs secuencia |
| 06 | Módulos ES6 | import/export, named vs default, re-exports, dynamic import |
| 07 | Error handling | tipos de error, try/catch/finally, errores custom, error boundaries |
| 08 | Iterators y Generators | Symbol.iterator, protocolo iterable, function*, yield |
| 09 | Proxy y Reflect | traps, validación, observabilidad, metaprogramación |
| 10 | Patrones funcionales | composición, currying, pipe, functor, inmutabilidad aplicada |

## Prerequisitos
Tener sólidos los basics: funciones, closures, arrays, objetos, scope.
