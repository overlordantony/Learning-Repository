# JavaScript / Advanced

El nivel donde JS deja de ser una herramienta y empieza a ser un lenguaje que entiendes de verdad.
Aquí se trabaja el motor, la memoria, la arquitectura y los patrones que separan el código que funciona
del código que está bien diseñado.

## Ejercicios

| # | Tema | Conceptos clave |
|---|------|-----------------|
| 01 | Event Loop Profundo | microtask/macrotask, starvation, scheduler, queueMicrotask |
| 02 | Memory y Performance | GC, memory leaks, WeakRef, profiling, optimización |
| 03 | Design Patterns | Observer, Strategy, Command, Decorator, Facade |
| 04 | Metaprogramación | Symbol, well-known symbols, Reflect, tagged templates |
| 05 | Concurrencia y Workers | Web Workers, SharedArrayBuffer, Atomics, postMessage |
| 06 | WebAssembly básico | qué es Wasm, cómo llamarlo desde JS, casos de uso |
| 07 | Arquitectura modular | DI, módulos desacoplados, event bus, plugin system |
| 08 | Testing avanzado | unit/integration/e2e, mocks, spies, testing async |
| 09 | Seguridad web | XSS, CSP, CSRF, sanitización, eval peligroso |
| 10 | Proyecto integración | mini-app que usa conceptos de todos los niveles |

## Prerequisitos
Dominar intermediate: clases, async/await, módulos, closures, prototypes.
